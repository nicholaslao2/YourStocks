03 — Page Relationship Summary

🔐 AUTH FLOW

Splash → Create Account → Risk Profile Setup → Dashboard

Splash → Login → 2FA / Biometric → Dashboard

🏠 DASHBOARD HUB

Central hub connecting to all 5 main feature areas via the bottom navigation bar and tappable holdings list.

📊 MARKETS FLOW

Markets & Watchlist → Stock Detail & Chart → Set Price Alert (→ Alerts screen)

🎮 SIMULATION FLOW

Simulated Trading → Simulation History. Uses real-time price data from Markets but executes on virtual balance only.

🔔 ALERTS FLOW

Alerts page ↔ Manage Alert Rules. Alerts also triggered from Stock Detail. Push & email notifications delivered via encrypted API.

🧠 RISK & ⚙️ SETTINGS

Risk Profile shows P&L analytics and asset allocation. Settings manages 2FA, biometrics, password, email, and sign out.
