04 · System Model

Input · Output · Transmission · Storage

How data flows through the YourStocks platform from collection to display.

⬇

Input

  • User credentials & preferences
  • Portfolio assets & quantities
  • Risk profile answers
  • Real-time market prices (API)
  • Financial news feeds
  • Simulation trading actions
⬆

Output

  • Live stock/crypto price display
  • Portfolio P&L dashboard
  • Price & watchlist alerts
  • Risk profile assessment
  • AI-generated insights
  • Simulated trading results
⇄

Transmission

  • Encrypted API calls (HTTPS/TLS)
  • Market data via WebSocket feeds
  • User auth via JWT tokens
  • Push notifications for alerts
  • Client-server encrypted sync
◫

Storage

  • User accounts & credentials
  • Portfolio & trade history
  • Risk profile responses
  • Alert preferences
  • Simulation session history
  • Custom dashboard layouts
05 · Requirements

Functional & Non-Functional

Functional Requirements

✓ Create & manage user accounts
✓ View and manage portfolio assets
✓ Real-time stock & crypto price tracking
✓ Price watchlists & push alerts
✓ Risk profile analysis & asset allocation
✓ Simulated trading mode (risk-free)
✓ AI-generated investment insights
✓ Tax reporting helper & recurring plans

Non-Functional Requirements

◆ Fast real-time data rendering
◆ End-to-end encryption for all data
◆ Accessible & intuitive UI
◆ High uptime & reliability (99.9%+)
◆ Biometric & 2FA security
◆ Scalable to many concurrent users
◆ Mobile-first responsive design
